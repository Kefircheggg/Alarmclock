# Часы с будильником  
Электронный будильник с дисплеем, клавиатурой, датчиком температуры, и пищалкой.  
Функционал:  
* Отображение даты;
* Отображение времени;  
* Отображение температуры;  
* Будильник с пищалкой, паролем и светодиодом;
